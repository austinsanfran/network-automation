# Network Automation

Python network automation scripts using Netmiko.

## Lab Environment
- GNS3 with VyOS router running in VMware Workstation
- Netmiko for SSH-based device interaction

## Scripts

### netmiko_test.py
Connects to a VyOS router via SSH and pulls interface information using 'show interfaces'.

**Requirements:**
- Python 3.11
- Netmiko (`pip install netmiko`)

## Goals
Building toward automated network configuration, auditing, and management as part of a CCNA to Network Automation career path.
